# Music Relax

Music Relax é um projeto de streaming de música desenvolvido por **Felippe Donatto**. Este projeto faz parte do seu portfólio profissional e demonstra habilidades avançadas em desenvolvimento frontend, UI/UX design e integração de áudio.

## 🚀 Tecnologias

- **Next.js 15**
- **React 19**
- **TypeScript**
- **Tailwind CSS**
- **Lucide React** (Iconografia)

## 🎨 Design & UX

O projeto foca em uma experiência de usuário premium, com:
- Interface em Dark Mode sofisticada.
- Navegação fluida e responsiva.
- Player de música interativo.
- Design focado em relaxamento e foco.
